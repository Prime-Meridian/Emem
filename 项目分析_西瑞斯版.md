## 项目概览

**项目名称**: Emem (喵喵机)  
**项目类型**: 仿微信界面的 AI 聊天 Web 应用  
**技术栈**: 原生 JavaScript + Dexie.js (IndexedDB) + PWA  
**代码规模**: 约 33,000 行  
**文件结构**:
- `index.html` (167KB) - 主页面结构
- `index_utf8.html` (1.7MB) - UTF-8 编码版本
- `scripts.js` (1.4MB, 33,105 行) - 核心逻辑
- `styles.css` (175KB) - 样式定义
- `新增总结.md` - 总结功能需求文档
- `架构文档.md` - 原有架构说明

### 4. 聊天系统 (Chat System)

#### 聊天类型
1. **单人聊天** (`type: 'single'`)
   - 1 个 AI 人设 + 1 个用户面具
   
2. **群聊** (`type: 'group'`)
   - 多个 AI 人设 + 1 个用户面具
   - AI 之间可以互动

#### 聊天数据结构
```javascript
{
    id: 'chat_xxx',
    name: '聊天名称',
    type: 'single' | 'group',
    personas: {
        ai: { name, avatar, content },  // AI 人设
        my: { name, avatar, content }   // 用户面具
    },
    history: [
        { 
            role: 'user' | 'assistant', 
            content: '...', 
            timestamp: xxx,
            isHidden: false  // 是否隐藏 (用于总结后的消息)
        }
    ],
    
    // 总结功能相关
    longTermMemory: [
        { content: '总结内容', timestamp: xxx }
    ],
    lastMemorySummaryTimestamp: xxx,  // 上次总结时间戳
    globalMemoryEnabled: false,       // 是否启用全局记忆
    
    settings: {
        enableAutoMemory: false,      // 是否启用自动总结
        autoMemoryInterval: 10,       // 自动总结间隔 (消息数)
        memoryWordThreshold: 500      // 总结字数阈值
    },
    
    isBlocked: false,
    lastMessage: '...',
    lastMessageTime: xxx
}
```

#### 核心聊天功能
- **消息发送**: `sendMessage()`
- **消息接收**: `receiveMessageHandler()`
- **消息编辑**: 长按消息 -> 编辑
- **消息引用**: 长按消息 -> 引用
- **消息多选**: 长按消息 -> 多选 -> 批量删除/转发
- **消息搜索**: `openSearchChatModal()`
- **清空记录**: `clearChatHistory()`
- **拉黑功能**: `blockContact()` / `unblockContact()`

### 5. 总结功能系统 (Summary System)

> 这是最新添加的核心功能，详见 `新增总结.md`

#### 功能概述
每个对话都有独立的"长期记忆"存储，类似世界书格式。当未总结的消息达到阈值时，自动触发总结，生成的内容追加到长期记忆末尾。

#### 核心函数

**手动触发总结** (第 6-11 行)
```javascript
async function handleManualSummary()
// 用户主动点击"总结"按钮时调用
```

**自动总结检查** (第 245-256 行)
```javascript
async function checkAndTriggerAutoSummary(chatId)
// 每次发送消息后检查是否达到阈值
```

**构建长期记忆上下文** (第 18-56 行)
```javascript
function buildLongTermMemoryContext(currentChat, currentChatId)
// 包含当前角色的长期记忆 + 其他角色的全局记忆
```

#### 总结提示词 (第 58-111 行)

**1. 对话总结提示词** (`DEFAULT_SUMMARY_PROMPTS.dialog`)
- 用于总结聊天记录
- 要求第三人称、过去时
- 输出 JSON 格式: `{"summary": "..."}`
- 使用变量: `${userNickname}`, `${chat.originalName}`

**2. 视频通话总结提示词** (`DEFAULT_SUMMARY_PROMPTS.call`)
- 用于总结视频通话记录
- 长度限制: 80 字以内
- 输出 JSON 格式

**3. 记忆精炼提示词** (`DEFAULT_SUMMARY_PROMPTS.refine`)
- 当长期记忆超过字数阈值时触发
- 整合并精炼现有记忆
- 消除冗余，保留关键信息

#### 破限功能 (第 196-239 行)
```javascript
function getJailbreakSuffix()
// 在总结提示词末尾添加破限文本
// 存储在 localStorage: 'jailbreakText', 'jailbreakEnabled'
```

#### 全局记忆功能
- 每个聊天可以选择是否启用"全局记忆"
- 启用后，其他角色可以读取该聊天的长期记忆
- 用于多角色联动场景
